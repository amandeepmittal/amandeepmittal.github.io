Open Source Stuff

I like open source stuff and here are some modules, packages, and bots that I have published:

  • @_100DaysOfCode A Twitter Bot to help developers participating in #100DaysOfCode. Inspired by @FreeCodeCamp. Twitter API
  • @nodejstweet An informative Twitter bot using Twitter API that retweets & favorites tweets relevant to MEAN Stack and Ionic Framework. Twitter API
  • atom-nodejs-snippets Nodejs and Javascript snippets package for Atom. Supports common ES6 syntax.
  • expressjs Expressjs, a Node.js framework, snippets package for Atom. Supports common ES6 syntax.
  • jade Jade/Pug snippets package for Atom.
  • StarTrek-Names Randomly Generate StarTrek characters.
  • StarTrek A Node.js module to insert inspiring StarTrek quotes into a site/application. Also supports command line view.
  • trimify allows an additional functionality over JavaScript native String method trim() such that you can trim whitespace(s) either in the beginning or the end of string and don't have to use Regular Expressions.

FreeCodeCamp Stuff

FreeCodeCamp Projects I have done:

  • Random StarTrek Quote Machine Web page displays random quotes. Twitter sharing option is available.
  • WikiPedia API Search Simple interface to the Wikipedia API that allows to search articles and display 10 possible article snippets for the user to select one. The snippets have an extract of the article, a title and when clicked, it will take you to the article. The app also has a random button for opening a random Wikipedia article. Wikipedia API
