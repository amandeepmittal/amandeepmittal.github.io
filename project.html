Open Source Stuff

I like open source stuff and here are some modules, packages, and bots that I have published:

  • @_100DaysOfCode A Twitter Bot to help developers participating in #100DaysOfCode. Inspired by @FreeCodeCamp & @ka11away
  • @BotWhoReads A Twitter Bot using Node.js, and Twitter API that retweets, follows, likes & recommend books to users tweeting #recommendbook
  • @nodejstweet A Twitter Bot using Twitter API that retweets & likes tweets relevant to MEAN Stack.
  • StarTrek-Names Randomly Generate StarTrek characters.
  • StarTrek A Node.js module to insert inspiring StarTrek quotes into a site/application. Also supports command line view.
  • atom-nodejs-snippets Nodejs and Javascript snippets package for Atom. Supports common ES6 syntax. Written in CoffeeScript
  • jade Jade snippets package for Atom. Written in CoffeeScript

FreeCodeCamp Stuff

FreeCodeCamp Projects I have done:

  • Random StarTrek Quote Machine Web page displays random quotes. Twitter sharing option is available. Using: HTML5, CSS, Bootstrap, Javascript, Jquery
  • WikiPedia API Search Simple interface to the Wikipedia API that allows to search articles and display 10 possible article snippets for the user to select one. The snippets have an extract of the article, a title and when clicked, it will take you to the article. The app also has a random button for opening a random Wikipedia article. Using HTML5, CSS, Javassript, Bootstrap, Jquery & bits of Wikipedia API
